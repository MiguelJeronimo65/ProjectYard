## Dicas de captura
- **Windows:** `Win`+`Shift`+`S` (recorte) → cola e guarda como PNG; ou `Alt`+`Print Screen` (só a janela).
- Usa o navegador em **ecrã largo** (≥ 1280 px) para a barra lateral e os cartões aparecerem bem.
- Para ecrãs com mais conteúdo, prefere **PNG** (texto nítido).
- Evita mostrar passwords ou dados sensíveis reais nas capturas.

> podes mantê-los e juntar as capturas reais, ou substituí-los.
